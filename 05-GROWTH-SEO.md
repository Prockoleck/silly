## SEO Strategy (Per Game Page)

Each game at `/games/[slug]` gets a dedicated SEO play:

### On-Page SEO

| Element | Rule |
|---|---|
| `<title>` | "Game Name — interactive fun by [Site Name]" |
| `<meta name="description">` | 1 sentence hook + 1 sentence outcome (max 160 chars) |
| `<meta property="og:title">` | Same as title |
| `<meta property="og:description">` | Same as description |
| `<meta property="og:image">` | 1200×630 PNG showing game preview |
| `<meta property="twitter:card">` | `summary_large_image` |
| `<meta name="robots">` | `index, follow` |

### Content Strategy

- **Game titles** use common, searchable words (e.g. "Spend a Billion" not "Moolah Mania")
- **Each game page** has 100–200 words of descriptive text below the game (hidden behind "About this game" expand)
- **Alt text** on all game thumbnails
- **Internal linking** — "Play another game" suggestions at bottom of each page
- **Sitemap.xml** auto-generated by Next.js

## Viral Mechanics (Built Into Games)

Every game must have at least one **shareable output**:

| Game | Shareable Output |
|---|---|
| Spend a Billion | "I spent $1B in 3 minutes — here's what I bought" card |
| Draw This Shape | Score card with drawing replay (animated GIF candidate) |
| Old Web Explorer | "I found this weird page from 1999" screenshot |
| Scale Shifter | "I collected 15/20 scale objects" progress card |
| Ranker | "My personality profile is [type]" image card |

Output = PNG image generated via canvas + downloadable + tweetable.

## Community & Social

### Reddit (highest ROI for these games)

| Subreddit | Post Type | Frequency |
|---|---|---|
| r/InternetIsBeautiful | Direct game link | Each new game |
| r/WebGames | Game submission | Each new game |
| r/SideProject | Dev story + game | Monthly |
| r/SampleSize | If game has data collection | When relevant |

**Reddit rules:** Engage with comments, don't spam, post from regular account (not new).

### Twitter / X

- Build in public: dev screenshots, game mechanics exploration
- Post each launch with a short video/screen recording
- Follow/engage with indie dev community

### TikTok / YouTube Shorts

- 15–30s clips of playing each game
- "Can you beat my score?" challenge format
- Link in bio
